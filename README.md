# WordPress

Connect one or more WordPress sites so agents can read posts, draft new ones, edit live articles, upload media, and run an in-CMS chat widget right inside the post editor. Edits to published posts use a demote-then-edit flow that keeps the previous live revision in WordPress's history.

## Capabilities

- Connect one or more WordPress sites to your workspace
- Browse and read recent posts on a connected site
- Create new draft posts and update existing ones from your agents
- Edit live posts without losing the previous published revision
- Upload images to a site's media library
- Run an in-CMS chat widget that makes edits to the current post inline
